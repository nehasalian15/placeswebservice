# Places Webservice
This webservice provides few places api features

## Steps	
	npm install
	npm start
	
## Prerequisite
	
	node 
	redis database on port 6379
## Database
		
	redis

## Features

	User management
	Place Data management
	Session management
	Log management
	Request Validation
	Error Management
	
## Api List
	baseUrl: http://localhost:3000/
	Users Api:
		user/create
		user/login
	Placee Api:
		place/autocomplete
		place/save
		place/find
